# HR Employee Attrition Dashboard 

## Project Overview
This project analyzes employee attrition data to understand the key factors that influence why employees leave an organization.
The goal was to practice **data cleaning, Excel analysis, pivot tables, and dashboard creation**.

## Tools Used
- Excel (Data Cleaning, Formulas, Pivot Tables)
- Tableau (Dashboard Visualization)
- GitHub (Project Showcase)

## Dataset
Source: Kaggle  
Sample HR Employee Attrition Dataset 

## Project Workflow

### Step 1: Data Cleaning
- Removed empty rows
- Converted order dates to proper date format
- Converted the Attrition column from Yes/No to a binary flag (1/0)
- Used EmployeeNumber as a unique key to remove duplicate records

### Step 3: Pivot Table Analysis
- Attrition vs Years at Company
- Attrition vs Salary
- Attrition by Department

### Step 4: Dashboard Creation
- Salary Attrition (Column Chart)
- Department Attrition (Bar Chart)
- Attrition Rate KPI (Text)

### Step 5: Insights
- Employees in lower salary bands show significantly higher attrition
- Attrition is highest among employees in their first 0â€“2 years
- Attrition decreases as salary and tenure increase
- Certain departments consistently experience higher turnover
